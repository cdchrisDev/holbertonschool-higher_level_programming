## Learning Objectives
* What is a superclass, baseclass or parent class
	* The class from which the subclass is derived
* What is a subclass
	* A class which inherits from a superclass
* How to list all attributes and methods of a class or instance
	* 1. To get the list of all the attributes, methods along with some inherited magic methods of a class, we use a built-in called dir().<br /> 2.
